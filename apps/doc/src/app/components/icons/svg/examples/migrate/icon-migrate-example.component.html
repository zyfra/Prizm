Migration Guide

  1. Для миграции с использованием прежних названий, вы можете использовать наш провайдер prizmIconsProvideOldSvgNameTransformer()
  2. Зарезирвируйте новый сет используемы иконок. Пример в текущем компоненте
  3. Вместо prizm-icons-svg добавьте и используйте prizm-icons или prizm-icons-full
